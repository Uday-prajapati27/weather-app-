# Weather App

A modern and responsive weather application built with HTML, CSS, and JavaScript. It allows users to search for a city and view weather details in a clean dashboard-style interface with animated UI components, theme switching, care recommendations, and weather trend visualization. [web:2][web:6]

## Overview

This project started as a basic weather app and was enhanced into a more professional frontend experience. The application focuses on usability, responsive design, smooth animations, and a more realistic weather dashboard layout inspired by modern weather apps. [web:6]

## Features

- Search weather by city name.
- Responsive and modern UI design.
- Dark mode and light mode switch.
- Animated weather card and interactive effects.
- Weather recommendation tips based on conditions such as summer heat, winter cold, rain, and humidity.
- Trend chart for past weather patterns.
- Keyboard-friendly search interaction.
- Loading and error state handling.
- Demo-ready frontend structure for future API integration. [web:2][web:6]

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js for graphical weather trends
- Open-Meteo API for weather and forecast data, which provides open-source weather access and does not require an API key for non-commercial use. [web:7][web:4]

## How to Run

1. Download or clone the project.
2. Open the project folder in your code editor.
3. Make sure all files are in the same folder.
4. Open `index.html` in your browser.

If the project uses live API calls, ensure you have an internet connection so weather data can load correctly. [web:2][web:7]

## How It Works

- The user enters a city name in the input field.
- JavaScript handles the search action and updates the UI dynamically.
- Weather information such as temperature and condition is displayed inside the weather card.
- Additional UI sections can show forecast details, recommendations, and weather trends.
- The interface adjusts for different screen sizes to improve the experience on desktop and mobile devices. [web:2][web:6]
